    • 【2023】如何在Github开源一个项目？(超详细文档说明)

    如果您完成了一个优秀或有趣的项目，并且您想把它开源在 GitHub 上与大众进行分享交流，那么这是一个很好的详细教程帮助您开源自己的首个项目。又或者您想使用 GitHub 进行项目版本管理或者进行团队开发，此文章也可帮助您将项目上传到自己的仓库中。因 GitHub 更新，此教程涉及到将本地 master 分支下代码上传到 GitHub 仓库的 main 分支的操作。

    涉及的浏览器的部分建议使用 Chrome 浏览器进行相关操作！
    在连接到 GitHub 的过程中可能有时会连接不上，请多次尝试！

    如果您已经注册有 GitHub 账号，那么您可以从“安装Git”章节开始看起；
    如果您已经注册有 GitHub 账号且电脑上已安装 Git，那么您可以直接从“上传项目致GitHub”章节看起。

    注册GitHub账号

    要将代码上传致 GitHub ，首先需要注册一个的 GitHub 账号。以下为具体步骤：

    • 首先百度搜索 GitHub 或者直接点击 https://GitHub.com/ 进入到 GitHub 官网。进入到首页后点击右上角的“Sign up”注册按钮进行注册，此时会跳转到注册页面。

    • 在注册页面根据提示一步步输入您的邮箱、密码、用户名，再选择是否接收邮箱通知（建议接收）。

      注册使用的邮箱简易用谷歌邮箱（当然，使用QQ邮箱或者163邮箱也行）。若提示邮箱已注册过账号或其它报错提示，请更改注册邮箱后重新进行注册。

    • 填写完以上信息后需要进行真人验证，按提示进行验证即可。验证完后点击“Create account”完成注册。
    • 接着会跳转到验证码验证界面。之前填写的注册邮箱会接收到一封验证邮件，将里面的验证码填写到验证框中即可。
    • 之后将跳转到问卷界面，按情况填写即可。填写完后会让我们选择用户类型为 Free 版本或 Pro 版本。其中，免费版是完全公开的，而私有版一般是由企业或者不愿公开仓库的个人用户购买，在这里，我们选择免费版就可以。（若您时学生，在此界面简易先选择免费版进行注册，若之后有需要再进行学生身份认证）
    • 完成以上步骤后便完成了注册的全过程，之后会自动跳转到您的个人主页。

    安装Git

    我们需要使用到 Git 来将项目提交到 GitHub ，Git 也是一个十分优秀的代码版本管理工具。（建议之后系统学习一下 Git 的使用方法，在以后使用 GitHub 以及团队开发时会经常使用。当然，就算您还未学习过 Git 也能够按教程步骤成功将项目开源到 GitHub。）以下为安装 Git 的具体步骤：

    • 首先百度搜索 Git 或者直接点击 https://git-scm.com/downloads 进入到 Git 官网，然后选择自己电脑对应的版本进行下载即可。

    • 下载好安装程序后双击运行。软件安装时根据个人习惯修改下安装路径，将 Git 安装到想要的路径下。只用修改安装路径，其它默认选项安装即可。
